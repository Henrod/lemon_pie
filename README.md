# LemonPie

![lint](https://github.com/Henrod/lemon_pie/actions/workflows/server.yml/badge.svg)

Judge your friends using emojis 😈

## Local development

### Start the server

```shell
cd server

# Start virtual environment
make venv
make dev/run
```

### Start the UI

```shell
cd client/lemon_pie
npm start
```
